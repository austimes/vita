# Feature Demo Examples

Focused v0.2 examples for individual public-surface concepts.

Notes:
- Legacy public demos for `demands`, `cases`, `constraints`, and `timeslices`
  were rewritten to the current v0.2 object model. The filenames are retained
  for catalog continuity, but they now demonstrate the nearest supported v0.2
  concept rather than the removed v0.1 surface.
- `example_with_facilities.veda.yaml` now demonstrates v0.2 `facilities`,
  multi-site membership, `technology_roles`, and explicit `transitions`.
- `example_with_trade.veda.yaml` demonstrates v0.2 `networks`.
- `example_with_bounds.veda.yaml` demonstrates stock plus `opportunities`
  (`max_new_capacity`) rather than legacy process-bound blocks.
- `example_with_timeslices.veda.yaml` now demonstrates temporal reference data
  via `temporal_index_series` and base-year stock adjustment.
