# Component Library Examples

Use this folder for examples that build systems from reusable component-library building blocks.

Temporal authoring in this folder should use canonical `time_series` declarations
and `quantity` / `adjust_to_base_year.series` references only.

Subfolders:
- `components/` — reusable standalone components
- `assemblies/` — composed systems built from components
- `studies/` — multi-case studies using component assemblies
